PHP基础学习(1)

发表于 2016-03-13   |   分类于 PHP   |  

学点PHP写服务器。

  1. PHP的脚本块以<?php开始，以?>结束，这是标准风格。

  2. echo和print是输出文本基础指令。（有区别） 

    <?php
    echo("hello,world");
?>

  3. 常量名以字符或下划线开头。
    大小写默认敏感。
    一般都大写。
    用define()函数设置常量值。

    <?php
    define("ABC","Hello,world");
    echo ABC;
//true表示对大小写不敏感
    define("HUB","Welcom",true);
    echo hub;
?>
  4. 预定义常量，如：PHP_OS

  5. 变量以$开头
    大小写敏感 

    <?php
    $txt = "Hello,world";
    $x = 5;
    $y = 10.5;
    echo $x+$y;
?>

  6. 变量的作用域
    1.局部变量：函数内部
    2.全局变量：所有函数以外；
    要在用户自定义函数内部使用，要用global关键字声明。

    <?php
    $zy = "你好";
    function lxt(){
        global  $zy;
        echo $zy;
    }
    lxt();
?>

    3.静态变量：在函数调用结束后仍保留变量值，用static声明。
    4.可变变量：在变量前再加一个“$”;允许动态改变一个变量名的名称，该变量的值由另外一个变量的值来确定。

    <?php
    $my_name = "myname";
    $myname = "yourname";
    echo $my_name;
    echo "<br>";
    echo $$my_name;
?>

  7. 数据类型：支持8种基本数据类型
    是弱类型语言，但有时仍需要用到类型转换。

    四种标量类型：boolean、integer、float、string
    两种复合类型：array(数组)、object(对象)
    两种特殊类型：resource(资源)、NULL(NULL)

    <?php
       $IsTrue = True;
    echo $IsTrue;
    //结果是1
?>
  8. “===”和“==”：前者表示不只是数值上相等，而且两者类型也一样。
  9. 连接字符串用“.”。

  10. 数组：两种方式声明

    <?php
  $arr = array("1"=>"汇","2"=>"智","3"=>"网");
    echo $arr[1]."<br>";

    $arr2 = array("汇","智","网");
    echo $arr2[0]."<br>";

    $arr3[1] = "汇";
    $arr3[2] = "智";
    $arr3[3] = "网";
    echo $arr3[1]."<br>";

    //用于输出数组结构
    print_r($arr3);
?>

    结果：

    汇
汇
汇
Array ( [1] => 汇 [2] => 智 [3] => 网 )

  11. 面向对象：

    构造方法_construct()；
    析构函数_destruct()：用于释放数据库资源等。

    <?php
  class car{
      public $color;
      public $size;

      public function _construct($x,$y){
          $this->color = $x;
          $this->size = $y;
      }

      public function printAttr(){
          echo "颜色：".$this->color."<br>";
          echo "大小：".$this->size."米";
      }
  }

    $bus = new car("blue","20");
    $bus->printAttr();
?>

结果：

颜色：
大小：米
