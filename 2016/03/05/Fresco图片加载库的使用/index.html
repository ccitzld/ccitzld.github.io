Fresco图片加载库的简单使用

发表于 2016-03-05   |   分类于 Android   |  

Fresco是FaceBook推出的一个图片加载库。

Fresco图片加载库不同于ImageLoader，它提供了控件级的图片加载。

如果你仅仅是想简单下载一张网络图片，在现在完成之前，显示一张占位图，那么简单地使用SimpleDraweeView即可。

1. 如果在Android studio中使用，加入compile ‘com.facebook.fresco:fresco:0.9.0+’即可引入。
2. 首先你需要加入网络权限，
3. 然后在Application初始化时，在应用调用setContentView()之前，进行初始化：
   //Application中：
Fresco.initialize(context);
4. 然后在xml布局文件中，加入控件的名称：
<com.facebook.drawee.view.SimpleDraweeView
    android:id="@+id/my_image_view"
    android:layout_width="20dp"
    android:layout_height="20dp"
    fresco:placeholderImage="@drawable/my_drawable"
/>
5. 开始加载图片：
Uri uri = Uri.parse("https://raw.githubusercontent.com/facebook/fresco/gh-pages/static/fresco-logo.png");
SimpleDraweeView draweeView = (SimpleDraweeView) findViewById(R.id.my_image_view);
draweeView.setImageURI(uri);

剩下的，Fresco会替你完成

  • 显示占位图直到加载完成
  • 下载图片
  • 缓存图片
  • 图片不再显示时，从图片中移除
